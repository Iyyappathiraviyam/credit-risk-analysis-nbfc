# 💳 Credit Risk Analysis & Fraud Detection (NBFC Use Case)

## 📊 Project Overview
This project focuses on analyzing customer credit behavior, identifying high-risk segments, and predicting loan default probability using Machine Learning techniques.

The objective is to simulate a real-world NBFC lending environment and support data-driven credit decisions.

---

## 🎯 Business Problem
NBFCs face challenges in:
- Identifying high-risk customers
- Reducing loan defaults
- Detecting fraudulent applications

This project builds a data-driven solution to:
✔ Improve risk assessment  
✔ Reduce financial losses  
✔ Support underwriting decisions  

---

## 🧠 Key Features
- Credit Risk Segmentation
- Loan Default Prediction Model
- Fraud Pattern Identification
- Customer Behavioral Analysis

---

## 🛠️ Tech Stack
- SQL (Data Extraction & Analysis)
- Python (Pandas, NumPy, Scikit-learn)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

---

## 📂 Dataset
- Simulated dataset based on:
  - Credit score (CIBIL-like)
  - Income & employment
  - Loan details
  - Repayment behavior

---

## 🔍 Exploratory Data Analysis
- Loan distribution analysis
- Default rate by income & credit score
- Correlation analysis

---

## 🤖 Model Building
- Logistic Regression
- Decision Tree / Random Forest

### Evaluation Metrics:
- Accuracy
- Precision / Recall
- ROC-AUC

---

## 📈 Key Insights
- Customers with low credit scores have higher default probability
- High loan-to-income ratio increases risk
- Certain behavioral patterns indicate potential fraud

---

## 💼 Business Impact
- Helps reduce default rate
- Improves credit approval strategy
- Supports risk-based pricing

---
